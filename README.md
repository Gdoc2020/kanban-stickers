# Kanban Stickers

Лёгкая open source Kanban-доска для Windows с локальным хранением данных. Проект распространяется по лицензии MIT.

Made and developed by **Technochip** — owned by **Grigory Bzhitov**.

## Возможности

- несколько досок и произвольные колонки;
- карточки с описанием, цветом, приоритетом и сроком;
- произвольные названия статусов и собственный цвет каждой карточки;
- пользователи с именами и email, назначение исполнителей задач;
- перетаскивание карточек между колонками;
- поиск, фильтры и архив;
- светлая и тёмная темы;
- режим «поверх остальных окон»;
- сворачивание в системный трей;
- автоматический запуск вместе с Windows;
- экспорт и импорт резервной копии.
- экспорт текущей доски в Microsoft Project XML с задачами, ресурсами и назначениями;
- интерфейс на русском, английском и турецком языках.

## Запуск для разработки

```powershell
npm.cmd install
npm.cmd start
```

## Создание установщика и portable-версии

```powershell
npm.cmd run dist
```

Готовые файлы появятся в папке `dist`.
