# Contributing to Kanban Stickers

Спасибо за интерес к проекту. Мы приветствуем исправления ошибок, улучшения интерфейса, новые языки и полезные интеграции.

## Начало работы

1. Сделайте fork репозитория и создайте отдельную ветку.
2. Установите зависимости командой `npm.cmd install`.
3. Запустите приложение командой `npm.cmd start`.
4. Перед отправкой изменений выполните `npm.cmd run check` и `npm.cmd test`.
5. Создайте pull request с кратким описанием изменений и способом проверки.

## Переводы

Все строки интерфейса находятся в объекте `I18N` файла `src/app.js`. Новый язык добавляется отдельным словарём и пунктом в списке `#language-select`.

## Принципы проекта

- данные пользователя по умолчанию остаются локальными;
- интерфейс должен оставаться быстрым и понятным;
- новые функции должны сохранять обратную совместимость с существующими досками;
- не добавляйте сетевую передачу данных без явного согласия пользователя.
